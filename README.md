Run Locally (Quick Start)

Install SQL Server and ensure it is running.
 
Clone the repository:

git clone https://github.com/3slam/CandidateManagement-API.git

Open the solution in Visual Studio OR navigate to the API folder using CLI.

Update the connection string in appsettings.json to point to your SQL Server.

Restore packages:

- dotnet restore
- dotnet run

API will start
